**O que é o Power Apps per app plan**

O Power Apps permite criar aplicativos empresariais personalizados sem a necessidade de programação complexa, utilizando recursos low-code e integrações com diversas fontes de dados.

O per app plan é indicado para cenários onde você quer disponibilizar um ou dois aplicativos específicos para um determinado grupo de usuários, pagando somente pelo acesso a esses aplicativos, sem precisar licenciar o uso ilimitado para cada colaborador.

Essa licença inclui:

* Acesso ao Dataverse com recursos premium.
* Uso de conectores premium como SQL Server, SAP, Salesforce e integrações via API.
* Execução de fluxos do Power Automate dentro dos apps.
* Suporte a aplicativos model-driven e canvas apps.

**Como funciona o licenciamento**

O per app plan é licenciado por usuário, por aplicativo. Na prática:

* Cada usuário licenciado pode acessar um aplicativo ou um portal Power Pages (dependendo da configuração).
* É possível adquirir mais de um per app plan para que o mesmo usuário acesse diferentes aplicativos.
* É ideal para soluções específicas que não exigem o acesso ilimitado que o per user plan oferece.

Exemplo real: Se a sua empresa precisa que o time de logística use um aplicativo de inspeção de frota e o time de vendas use um CRM interno, você pode adquirir licenças per app plan apenas para os colaboradores que utilizarão cada um desses aplicativos, reduzindo custos e mantendo acesso a funcionalidades premium.

**Benefícios estratégicos**

* **Custo otimizado:** Pague apenas pelo que realmente vai ser usado. É perfeito para empresas que precisam de automações e integrações premium em poucos aplicativos específicos.
* **Acesso a recursos premium:** Inclui conectores premium, Dataverse e recursos de segurança avançada para aplicativos corporativos.
* **Integração com outros serviços Microsoft:** Integra-se nativamente com SharePoint, Teams, Dynamics 365, Power Automate e Power BI.
* **Flexibilidade no desenvolvimento:** Permite criar tanto canvas apps (apps personalizados com interface livre) quanto model-driven apps (apps baseados em dados e lógica de negócios do Dataverse).
* **Escalabilidade gradual:** Comece com um ou dois aplicativos e, se necessário, evolua para licenças per user plan quando o uso crescer.

**Aplicações práticas**

* Comercial: CRM simplificado para acompanhamento de leads e oportunidades.
* Recursos Humanos: app para gestão de solicitações de férias, ausências e onboarding.

**What is the Power Apps per app plan**

Power Apps allows you to build custom business applications without complex coding, using low-code resources and integrations with multiple data sources.

The per app plan is designed for scenarios where you want to give users access to one or two specific applications, paying only for those apps rather than licensing unlimited use for every employee.

This license includes:

* Access to Dataverse with premium capabilities.
* Use of premium connectors such as SQL Server, SAP, Salesforce, and API integrations.
* The ability to run Power Automate flows within apps.
* Support for both model-driven and canvas apps.

**How the licensing works**

The per app plan is licensed per user, per app. In practice:

* Each licensed user can access one app or one Power Pages portal (depending on configuration).
* You can purchase more than one per app plan for the same user to allow access to different applications.
* It’s ideal for targeted solutions that don’t require the unlimited access of a per user plan.

Real example: If your company needs the logistics team to use a fleet inspection app and the sales team to use an internal CRM, you can purchase per app plan licenses only for the employees who will use each app, reducing costs while maintaining access to premium features.

**Strategic benefits**

* **Optimized cost:** Pay only for what will actually be used. Perfect for companies that need premium automation and integration in just a few specific applications.
* **Access to premium features:** Includes premium connectors, Dataverse, and advanced security features for corporate applications.
* **Integration with other Microsoft services:** Seamlessly integrates with SharePoint, Teams, Dynamics 365, Power Automate, and Power BI.
* **Flexibility in development:** Allows you to build both canvas apps (custom apps with free-form design) and model-driven apps (data-driven apps built on Dataverse business logic).
* **Gradual scalability:** Start with one or two apps, and upgrade to per user plan licenses when usage expands.
